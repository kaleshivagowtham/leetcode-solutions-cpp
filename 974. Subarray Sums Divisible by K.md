# 974. Subarray Sums Divisible by K
---

# Intuition
Prefix sum

# Complexity
- Time complexity: O(n)

- Space complexity: O(n+1)

# Code
```
class Solution {
public:
    int subarraysDivByK(vector<int>& nums, int k) {
        
        int count=0,i=0,n=nums.size(),sum=0;
        map<int,int> store;
        store.insert({0,1});
        while(i<n) {
            sum += nums[i];
            int rem = sum%k >= 0 ? sum%k : k + ( sum % k);
            if( store.find(rem) != store.end())
                count += store[rem];
            store[rem]++;
            i++;
        }
        return count;
    }
};
```
